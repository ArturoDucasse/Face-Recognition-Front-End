# Face-Recognition-Front-End

# Description
Website that enables the user to detect faces on the photos he provides.

# Prerequisites
- clarifai

```npm i clarifai```
- react
 
```npm i react```
- react-dom

```npm i react-dom```
- react-particles-js
 
```npm i react-particles-js```
 - react-scripts
  
```npm i react-scripts```
- react
 
```npm i react```
- react-tilt

```npm i react-tilt```
- tachyons

```npm i tachyons```
- web-vitals

```npm i web-vitals```

# Installation
```
npm install 
```
```
npm start
```
Staring the developming mode
```
npm run develop
```
Testing
```
npm test
```
Create build
```
npm run build
```
